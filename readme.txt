PDF 多選區裁切工具 使用說明
===========================
設計者：DY

【功能簡介】
本程式可開啟 PDF 檔案，自由選擇多個裁切區域並批次輸出成固定尺寸 (10x15cm) 的 PDF 檔案。每頁可加入底部提示圖片。

【使用步驟】
1. 開啟 PDF：
   - 點選「開啟 PDF」，選擇檔案。

2. 選擇裁切區：
   - 用滑鼠拖曳畫出紅色框，會自動編號。
   - 如需重新畫選，請點「清除選取區」或「返回上一步」。

3. 頁面操作：
   - 使用「上一頁」/「下一頁」切換 PDF 頁面。
   - 可勾選「批次多頁處理」將所有頁面的同一位置裁切。

4. 匯出結果：
   - 點「匯出裁切區」產生新 PDF。
   - 若裁切區為空白將不產生頁面。
   - 匯出的每頁將插入「裁切後加入的圖.jpg」。

5. 新增流水號：
   - 可自訂流水號 起始碼。
   - 裁切後會按 起始碼 自動編流水號 於右上角。
   

【注意事項】
- 請確保程式目錄內有放置 `裁切後加入的圖.jpg`。
- 如 PDF 有密碼，預設會嘗試密碼 `66608251`，不成功則會要求手動輸入。

【建議使用環境】
- Python 3.8+
- 安裝套件：PyMuPDF、Pillow、tkinter（標準內建）
